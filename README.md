# RockPaperScissors

Download repository  
Build Tool: Maven

### Build and run application
Change to directory RockPaperScissorsGame containing the .pom file and execute in command line:
```
mvn package
```
Then execute:
```
java -cp target/RockPaperScissorsGame-1.0-SNAPSHOT.jar RockPaperScissors.App
```

### Run Tests
Change to directory RockPaperScissorsGame containing the .pom file and execute in command line:
```
mvn test
```
